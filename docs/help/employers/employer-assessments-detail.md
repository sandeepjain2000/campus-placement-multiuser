# Assessment CSV workflow (step by step)

> **Section:** Employers  
> **Source:** help  
> **Audience:** employer

Use this when you need to record round outcomes for many applicants at once.

1. Open Assessment map and save round labels for each opportunity type you use.
2. Open Assessment uploads (CSV) and pick the tab (Internship, Jobs, Drive, or Projects).
3. Click Export CSV — the file lists all applications on that tab with columns system_id, college_roll_no, placement_drive_id, job_id, round_1…round_5, remarks, and related fields.
4. Edit round columns offline (do not rename headers). Save as CSV.
5. Upload the file on the same tab. Review accepted vs rejected rows in upload history.
6. Confirm results on Hiring Results Dashboard. The college sees the same data on Hiring Assessment (read-only).

Alternative: use Assessment Update Online on the same tabs to edit round cells in the browser and Save — no file required.

Assessment updates do not send email to students; they appear in dashboard views only.
