# Assessments, interviews, applications, offers

> **Section:** Employers  
> **Source:** help  
> **Audience:** employer

Assessment map — configure display names for round_1…round_5 per opportunity type (Internship, Jobs, Drive, Projects). CSV upload and Assessment Update Online use these labels automatically.

Assessment uploads (CSV) — tabbed by opportunity type. Use Export CSV on each tab to download all applications (same columns for re-upload). Upload CSV with no column-mapping dialog; each row targets a job, drive, or program via job_id / placement_drive_id.

Assessment Update Online — edit round results inline in a tabbed application table (alternative to export → edit → upload).

Hiring Results Dashboard — read-only consolidated view of assessment outcomes by tab; export available. Employers do not edit rows here.

Interview Scheduling — propose or confirm slots; students see updates on their side.

Applications — review the candidate pipeline with filters.

Offers — record outcomes individually. Bulk offers CSV opens from the Offers page (link on that screen); it is not a separate sidebar menu item.
