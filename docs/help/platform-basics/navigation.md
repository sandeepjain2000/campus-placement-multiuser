# Navigation, home, and sidebar

> **Section:** Platform basics  
> **Source:** help  
> **Audience:** all

After sign-in you land on your role home (student / employer / college / admin hub). Use the sidebar to move between sections; on smaller screens, open the menu from the top bar.

The full-screen hub lists every destination in grouped columns. Your last-used section is remembered. Use the in-dashboard search (where available) to jump to a screen by name.
